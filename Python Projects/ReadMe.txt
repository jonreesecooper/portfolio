PYTHON PROJECTS

A collection of projects created using Python.

Projects:
Checkbook
File Transfer GUI
Student Tracking App
Website Generator

Checkbook:
Web application created using Django that allows users to create a checkbook and provides CRUD capabilities
for the user's checking account.

File Transfer GUI:
Basic Graphic User Interface created using tkinter module that allows users to search for all text files in a particular folder
on their computer, search for any updated within the past 24 hours, and transfer them to a a new
folder.

Student Tracking App:
Graphic User Interface created using tkinter module that allows the user to leverage a database of students
and provides basic CRUD functionality.

Website Generator:
Graphic User Interface creatred using tkinter module that allows user to spin up and launch a basic HTML
website with a custom h2 tag displaying user provided text.
